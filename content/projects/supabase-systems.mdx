# Supabase Backend Systems

This project demonstrates a production-grade backend architecture built on the Supabase platform. It features a complex PostgreSQL schema with full row-level security (RLS), real-time capabilities, and automated CI/CD pipelines.

## Key Features

- **Complex Schema**: A 14-table PostgreSQL database with intricate relations and constraints.
- **Row-Level Security**: Granular access control for every table to ensure data integrity and security.
- **Edge Functions**: Serverless logic implemented via Deno Edge Functions for high-performance computing.
- **Real-time Engine**: Real-time data synchronization across all clients.
- **Automated CI/CD**: Full deployment pipeline using GitHub Actions.

## Technical Stack

- **Database**: PostgreSQL (via Supabase)
- **Functions**: Deno
- **Security**: JWT + refresh tokens + RLS
- **Operations**: GitHub Actions
