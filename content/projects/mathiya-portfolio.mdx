# MATHIYA Portfolio

This portfolio is the ultimate showcase of modern web engineering, combining Next.js 15, React 19, and advanced 3D graphics to create a premium developer experience. It features a custom robot mascot named **BYTE**, 16 unique GLSL shaders, and 6 playable games.

## Key Features

- **Next.js 15 + React 19**: Leveraging the latest stable features for concurrent rendering and server-side performance.
- **BYTE R3F Mascot**: A reactive 3D/SVG mascot with full IK eye, arm, and head tracking.
- **16 GLSL Shaders**: Custom-built shaders for everything from background fields to interactive card holograms.
- **6 Playable Games**: Fully functional arcade games integrated directly into the React architecture.
- **Premium Design**: A "Linear x Vercel x Apple" level dark aesthetic with smooth transitions and cinematic loaders.

## Technical Stack

- **Framework**: Next.js 15 (App Router)
- **Animation**: Framer Motion 11 + GSAP
- **3D/GL**: React Three Fiber + raw GLSL
- **State**: Zustand + SWR

> "Building the MATHIYA portfolio was an exercise in pushing the boundaries of what is possible in a static Next.js site."
