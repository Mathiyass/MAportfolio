# Android Handheld Companion

The Handheld Companion is a comprehensive Android application developed for the JIAT HHDPII coursework. It serves as a multifunctional tool for mobile users, integrating sensor data, local persistence, and remote API connectivity.

## Key Features

- **Sensor Integration**: Real-time tracking and visualization of device sensors (accelerometer, gyroscope).
- **Offline First**: Robust local storage using Room database and DAO patterns.
- **API Connectivity**: Seamless remote data fetching via Retrofit.
- **Modern UI**: Follows Material Design 3 guidelines for a clean and intuitive user interface.
- **Lifecycle Management**: Implements ViewModel and LiveData for architecture stability.

## Technical Architecture

- **Language**: Kotlin
- **Database**: Room
- **Networking**: Retrofit
- **Testing**: JUnit + Espresso
