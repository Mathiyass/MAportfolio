# MA_Dev Marketplace

The MA_Dev Marketplace is a high-performance native Android m-commerce platform designed specifically for software developers. Often described as **"Steam for Developers"**, it provides a centralized hub for buying, selling, and managing developer assets, from code snippets and libraries to full-scale UI kits and 3D models.

## Key Features

- **Native Android Performance**: Built with Kotlin and following Clean Architecture (MVVM) principles for a buttery-smooth user experience.
- **Supabase Backend**: Utilizes Supabase for real-time PostgreSQL database management, row-level security (RLS), and secure file storage.
- **AI-Driven UI**: 18 screens designed using Google Stitch AI, featuring a premium dark aesthetic with cyan and red neon accents.
- **Secure Payments**: Integrated with PayHere payment gateway for seamless and secure transactions.
- **Developer Hub**: Seller verification, asset moderation, and a robust review/rating system to ensure high-quality content.

## Technical Architecture

The application follows a modern mobile architecture stack:
- **Language**: Kotlin
- **Persistence**: Room for local caching + Supabase PostgreSQL for cloud sync.
- **Networking**: Retrofit + Coroutines for asynchronous API calls.
- **Real-time**: Firebase FCM for notifications and Supabase Realtime for download tracking.
- **UI**: Material Design 3 + Custom custom components for the MATHIYA brand feel.
