# Killer Quebes

Killer Quebes is a high-octane 3D arcade shooter built using C++ and the TL-Engine (TLX). It features a fast-paced survival experience where players must defend against waves of increasingly difficult enemies while managing score combos and power-ups.

## Key Features

- **3D Arcade Mechanics**: Utilizes AABB collision detection and billboard particle effects for explosive combat.
- **Progressive Difficulty**: Wave-based state machine that scales enemy density and speed as the player progresses.
- **Score Combo System**: Encourages precise shooting and rapid kills to maximize points.
- **Persistent Data**: High score system with file I/O for saving and loading player progress.

## Technical breakdown

- **Language**: C++
- **Graphics Engine**: TL-Engine
- **Physics**: Custom AABB collision and particle systems.
- **Logic**: Wave state machine for game flow management.
