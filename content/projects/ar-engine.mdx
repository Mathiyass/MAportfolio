# AR Experience Engine

The AR Experience Engine is a cutting-edge spatial computing platform built on WebXR and ARCore. It enables users to view and interact with 3D models in their physical environment directly through a mobile browser.

## Key Features

- **Browser-Based AR**: No app installation required, leveraging the WebXR Device API.
- **Spatial Tracking**: Supports both marker-based and markerless tracking for maximum flexibility.
- **Interactive Placement**: Tap-to-place mechanics for positioning 3D assets in real-world space.
- **High-Performance Rendering**: Built on WebGL2 for smooth, realistic 3D visuals.

## Technical breakdown

- **APIs**: WebXR, ARCore
- **Rendering**: WebGL2
- **Tracking**: DeviceOrientation + ARCore plane detection
- **Platform**: Cross-browser mobile support
