How it was built.

The technical architecture behind this portfolio.

Zero Frameworks. Absolute Control.

This portfolio was built entirely without external JavaScript frameworks or CSS libraries. No React, no Vue, no Tailwind, no Three.js. It is a testament to the power of raw WebGL2, Vanilla ES6 modules, and modern CSS features like custom properties and CSS grid.

By bypassing abstraction layers, I retained absolute control over the render loop, enabling a sustained 60fps across complex WebGL shaders, particle simulations, and a fully interactive robot mascot.

BYTE: The Robot Mascot

BYTE is rendered entirely via a single inline SVG. His eyes follow the cursor using Inverse Kinematics (IK), calculating the angle and distance to the pointer via Math.atan2 and Math.hypot, and lerping the iris positions.

His mouth expressions are achieved through SVG path d-attribute interpolation. He possesses 15 unique greetings based on the current page context, and reacts to events like high-velocity cursor movements (getting startled) or inactivity (falling asleep).

WebGL2 & Ping-Pong FBO

The 16 shaders featured across the site (including the background atmospheric effects) are driven by a custom lightweight WebGL2 engine. The engine compiles the GLSL, binds a full-screen quad, and updates uniforms on every requestAnimationFrame.

Advanced effects, such as the GPU particle system (SHADER_06), utilize a ping-pong Framebuffer Object (FBO) technique. This allows the shader to read from the previous frame's texture to compute physics, and write the updated positions to the next frame's texture, alternating the buffers seamlessly.

Audio Engine

The ambient background sound and UI interaction feedback are generated purely via mathematics using the Web Audio API. No MP3s or WAVs are loaded. A 40Hz sine wave drone coupled with an 80Hz triangle harmonic creates the persistent atmospheric hum, while short-envelope sine waves provide the interaction pings.
