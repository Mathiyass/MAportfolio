Architecture

Built for Scale.

The platform is built on a robust architecture leveraging Kotlin's modern features. The MVVM pattern ensures separation of concerns, while Coroutines and Flow handle asynchronous operations smoothly.

Backend operations are powered by Supabase, providing real-time PostgreSQL database access, Row Level Security (RLS) for data protection, and Edge Functions for secure payment processing via PayHere webhooks.
